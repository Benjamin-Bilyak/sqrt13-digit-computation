# Computation of √13 to [X] Decimal Digits

This repository documents Benjamin Bilyak’s computation of the square root of 13, √13, to ten billion decimal digits using y-cruncher.

The purpose of this project was to carry out and publish an independent high-precision computation of an irrational square root, including the computed digits and supporting information. The project was also motivated by the limited availability of publicly accessible information on large-scale computations of √13.

## Project Details

- Constant: √13 / sqrt(13) / square root of 13
- Decimal digits computed: 10,000,000,000
- Computed by: Benjamin Bilyak 
- Software: y-cruncher
- Date completed:  Friday May 29 2026
- Archive: (https://zenodo.org/records/20443567?)
